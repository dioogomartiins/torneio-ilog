⚽ Formato do torneio

Define aqui o formato base do teu torneio.

1 = cada equipa joga uma vez · 2 = casa e fora · 3+ = repetição configurável

🏆 Eliminatórias (Mata-Mata)

Ativa o modo de eliminatórias que será jogado após o fim da fase de liga.

📐 Pontuação

Exemplo: Porto 4–1 Benfica com vitória=3 e bónus=1 a partir de 3 golos marcados → Porto fica com 3+1 = 4 pontos.

👥 Equipas (Cores)

Muda os nomes e escolhe a cor principal da equipa livremente.

👕 Gestão de Plantéis

Seleciona uma equipa e associa jogadores da base de dados para acompanhar a tabela de melhores marcadores.

Se um jogador ainda não existe, cria-o primeiro na aba 👤 Jogadores.

👤 Base de Dados de Jogadores

Regista jogadores e define os seus atributos. Podes atribuir cada jogador a uma equipa do torneio.

🎯 Snake Draft

Define os nomes das equipas, seleciona os jogadores presentes e clica em Fazer Draft para gerar equipas equilibradas automaticamente.

🏆 Classificação

Critérios de desempate, por ordem: pontos → diferença de golos → golos marcados → confronto direto (mini-tabela só com os jogos entre as equipas empatadas) → menos golos sofridos → ordem alfabética.
